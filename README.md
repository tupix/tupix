# tupix

tupix is a simple work-in-progress OS targeted for the Raspberry Pi 2 Model B
(ARMv7-A). \
It is primarily meant as an educational project to learn the basics of OS
development.

Currently following components are implemented:
 - Interrupt Handling
 - Threads & Processes with Scheduler
 - Kernel & User Space (incl. memory protection through MMU)
 - Systemcalls
